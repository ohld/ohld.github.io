Предсказания: локальные LLM на Apple чипах и новые LAN-party

Предсказания про AI: Apple выпускает чипы для локальных LLM, большинство задач делается оффлайн, возрождаются LAN-party с общим LLM compute.

Заскриньте:

Apple выпустит чипы для локальных LLM

Через N месяцев Apple выпускает новый чип, позволяющий запускать локальные LLM достаточно умные для базовых задач с нужной скоростью tokens/second. Китайские компании не отстают.

Большинство AI-задач уйдут в офлайн

Большинство задач (управление браузером, файлами, поиск) теперь происходят локально. Для более умных задач — возможно, подписка Apple Plus с анонимностью, доп. безопасностью и доступом к frontier моделям.

LAN-пати возвращаются

LAN-пати возрождаются, но в новой форме: раньше ты нёс свою оперативку к другу, чтобы запускать игры с лучшей графикой и FPS. Теперь — несёшь свой LLM compute, чтобы играть в новый вид игр. Вместе. 🥤
