# Советы от создателя Claude Code: git worktrees, self-improving CLAUDE.md и ревью

Source: https://okhlopkov.com/sovety-sozdatel-claude-code-git-worktrees/

Боря, создатель Claude Code, скинул тред с типсами по клоду Полезного много, но вот что я хочу попробовать: Git Worktrees: 3-5 параллельных Claude сессий Каждый ворктри = отдельная ветка и свой Claude. Самый большой анлок по мнению всей команды. У меня сейчас сетап одного проекта со всеми спеками + ссылки на папки с кодом — не очень best practice. Self-improving CLAUDE.md После каждой ошибки говоришь: "обнови свой CLAUDE.md чтобы больше так не делать". Claude, говорят, хорошо пишет правила для самого себя. Claude как строгий ревьюер Не только "расспроси меня обо всем", но и "докажи мне, что это сработает". /techdebt команда Собери себе скилл, чтобы запускать в конце каждой сессии: найди дублированный код, закоммить, обнови спеку.
